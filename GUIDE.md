## المقدمة والتثبيت

### ما هي هافنيكس؟

هافنيكس هي لغة برمجة تستخدم اللهجة السودانية لتنفيذ الأوامر. تم تصميمها لتكون سهلة الفهم والاستخدام لمن يتحدثون العربية، وخاصة باللهجة السودانية.

### المتطلبات

- Python 3.8 أو أحدث

### التثبيت

```bash
# تنزيل المشروع
git clone https://github.com/Snixrs/Havnix-Language.git
cd Havnix-Language

# تثبيت المتطلبات (اختياري - فقط إذا أردت MySQL أو API)
pip install -r requirements.txt
```

### التشغيل

```bash
# تشغيل ملف هافنيكس
python havnix.py main.havnix

# عرض المساعدة
python havnix.py --مساعدة

# عرض رقم الإصدار
python havnix.py --نسخة
```

## المتغيرات والأنواع

### تعريف المتغيرات

كل متغير يبدأ بعلامة `$` ويُنهى بفاصلة منقوطة `;`

```havnix
// نص
$الاسم = "عثمان";

// رقم صحيح
$العمر = 18;

// رقم عشري
$الطول = 175.5;

// قيمة منطقية
$سوداني = صاح;
$أجنبي = غلط;

// قائمة (مصفوفة)
$الألوان = ["أحمر", "أخضر", "أزرق"];

// قاموس (كائن)
$شخص = {"الاسم": "أحمد", "العمر": 25};

// قيمة فارغة
$فارغ = فاضي;
```

### الثوابت

```havnix
ثابت $PI = 3.14159;
ثابت $اسم_التطبيق = "تطبيقي";
// لا يمكن تغيير قيمة الثابت بعد تعريفه
```

### الوصول لعناصر القائمة

```havnix
$فواكه = ["تفاح", "موز", "برتقال"];
قول ليهو("$فواكه[0]");  // تفاح
قول ليهو("$فواكه[1]");  // موز
```

### الوصول لقيم القاموس

```havnix
$شخص = {"الاسم": "أحمد", "العمر": 25};
$مفاتيح = مفاتيح($شخص);
$قيم = قيم($شخص);
```

### تحويل الأنواع

```havnix
$رقم = رقم("42");        // تحويل نص لرقم صحيح
$عشري = عشري("3.14");    // تحويل نص لرقم عشري
$نصي = نص(42);           // تحويل رقم لنص
$منطقي = منطقي(1);       // تحويل لقيمة منطقية
$النوع = نوع($رقم);      // معرفة نوع المتغير
```

### أنواع البيانات

| النوع | المثال | الوصف |
|-------|--------|-------|
| نص | `"مرحبا"` | نص (String) |
| رقم | `42` | عدد صحيح (Integer) |
| عشري | `3.14` | عدد عشري (Float) |
| منطقي | `صاح` / `غلط` | قيمة منطقية (Boolean) |
| قائمة | `[1, 2, 3]` | مصفوفة (Array/List) |
| قاموس | `{"مفتاح": "قيمة"}` | كائن (Dictionary/Object) |
| فاضي | `فاضي` | قيمة فارغة (Null/None) |

```havnix
$أرقام = [1, 2, 3];
```

```havnix
// طلب إدخال من المستخدم
اسأل($الاسم, "ما اسمك؟ ");

// صيغة بديلة
$العمر = اسأل("كم عمرك؟ ");
```

## العمليات الحسابية

### العمليات الأساسية

```havnix
$أ = 10;
$ب = 3;

$جمع = $أ + $ب;     // 13
$طرح = $أ - $ب;     // 7
$ضرب = $أ * $ب;     // 30
$قسمة = $أ / $ب;    // 3.333...
$باقي = $أ % $ب;    // 1
```

### المعاملات المنطقية

```havnix
// و (AND)
لو ($العمر >= 18 و $سوداني == صاح) {
    قول ليهو("مواطن سوداني بالغ");
}

// أو (OR)
لو ($العمر < 13 أو $العمر > 65) {
    قول ليهو("إعفاء من الرسوم");
}

// مو (NOT)
لو (مو $متخرج) {
    قول ليهو("لسه طالب");
}
```

### معاملات المقارنة

| المعامل | المعنى |
|---------|--------|
| `==` | يساوي |
| `!=` | لا يساوي |
| `>` | أكبر من |
| `<` | أصغر من |
| `>=` | أكبر من أو يساوي |
| `<=` | أصغر من أو يساوي |

## الجمل الشرطية

### لو (if)

```havnix
لو ($العمر >= 18) {
    قول ليهو("كبير");
}
```

### لو / غير كدا (if/else)

```havnix
لو ($العمر >= 18) {
    قول ليهو("كبير");
}
غير كدا {
    قول ليهو("صغير");
}
```

### لو / غير كدا لو / غير كدا (if/elif/else)

```havnix
لو ($الدرجة >= 90) {
    قول ليهو("ممتاز");
}
غير كدا لو ($الدرجة >= 80) {
    قول ليهو("جيد جداً");
}
غير كدا لو ($الدرجة >= 70) {
    قول ليهو("جيد");
}
غير كدا لو ($الدرجة >= 60) {
    قول ليهو("مقبول");
}
غير كدا {
    قول ليهو("راسب");
}
```

### ولو (elif بديل)

```havnix
لو ($لون == "أحمر") {
    قول ليهو("🔴");
}
ولو ($لون == "أخضر") {
    قول ليهو("🟢");
}
ولو ($لون == "أزرق") {
    قول ليهو("🔵");
}
غير كدا {
    قول ليهو("لون غير معروف");
}
```

## الحلقات التكرارية

### طالما (while)

```havnix
$عداد = 1;
طالما ($عداد <= 10) {
    قول ليهو("العدد: $عداد");
    $عداد = $عداد + 1;
}
```

### لكل (foreach)

```havnix
$الأسماء = ["عثمان", "أحمد", "محمد", "علي"];
لكل $اسم في $الأسماء {
    قول ليهو("مرحبا يا $اسم!");
}
```

### تكرار (for)

```havnix
تكرار ($ت = 0; $ت < 10; $ت = $ت + 1) {
    قول ليهو("التكرار رقم: $ت");
}
```

### وقف (break) و كمل (continue)

```havnix
// وقف - إيقاف الحلقة
$ع = 1;
طالما ($ع <= 100) {
    لو ($ع > 5) {
        وقف;
    }
    قول ليهو("$ع");
    $ع = $ع + 1;
}

// كمل - تخطي بقية التكرار الحالي
تكرار ($ت = 1; $ت <= 10; $ت = $ت + 1) {
    $باقي = $ت % 2;
    لو ($باقي != 0) {
        كمل;
    }
    قول ليهو("رقم زوجي: $ت");
}
```

## الدوال

### تعريف دالة

```havnix
دالة سلام(الاسم) {
    قول ليهو("مرحبا يا $الاسم!");
}
```

### استدعاء دالة

```havnix
جيب لي سلام("عثمان");
// أو
سلام("عثمان");
```

### دالة مع قيمة مرجعة

```havnix
دالة جمع(أ, ب) {
    $نتيجة = أ + ب;
    رجع $نتيجة;
}

$المجموع = جمع(10, 20);
قول ليهو("المجموع: $المجموع");
```

### دالة بدون معاملات

```havnix
دالة تحية() {
    قول ليهو("السلام عليكم!");
}

جيب لي تحية();
```

## معالجة الأخطاء

### جرب / امسك (try/catch)

```havnix
جرب {
    $نتيجة = 10 / 0;
    قول ليهو("النتيجة: $نتيجة");
}
امسك ($خطأ) {
    قول ليهو("حصل خطأ: $خطأ");
}
```

### جرب / امسك / واخيراً (try/catch/finally)

```havnix
جرب {
    // كود قد يحصل فيه خطأ
    $بيانات = اقرأ_ملف("ملف.txt");
}
امسك ($خطأ) {
    قول ليهو("خطأ: $خطأ");
}
واخيراً {
    // هذا الكود يتنفذ دايماً
    قول ليهو("انتهت المحاولة");
}
```

### بديل: لو_فشل / دايماً

```havnix
جرب {
    // كود
}
لو_فشل ($خطأ) {
    قول ليهو("فشل: $خطأ");
}
دايماً {
    قول ليهو("تنظيف...");
}
```

## العمليات على النصوص

### الدوال المتوفرة

```havnix
$نص = "مرحبا بكم في هافنيكس";

// طول النص
$الطول = طول($نص);              // عدد الحروف

// قطع جزء من النص
$جزء = قطع($نص, 0, 5);          // "مرحبا"
$بقية = قطع($نص, 6);            // "بكم في هافنيكس"

// استبدال
$جديد = استبدل($نص, "هافنيكس", "Havnix");

// تقسيم لكلمات
$كلمات = قسم($نص, " ");         // ["مرحبا", "بكم", "في", "هافنيكس"]

// أحرف كبيرة/صغيرة
$كبير = كبر("hello");           // "HELLO"
$صغير = صغر("HELLO");           // "hello"

// هل يحتوي؟
$موجود = يحتوي($نص, "هافنيكس");  // صاح

// ضم قائمة لنص
$أسماء = ["أحمد", "محمد", "علي"];
$مضموم = ضم($أسماء, ", ");       // "أحمد, محمد, علي"

// عكس النص
$معكوس = عكس("مرحبا");          // "ابحرم"
```

```havnix

لكل $مفتاح في مفاتيح($شخص) {
```

## التعامل مع APIs

### المتطلبات

```bash
pip install requests
```

### طلب GET (جلب بيانات)

```havnix
$رد = جيب_من("https://api.example.com/users");
قول ليهو("الحالة:");
اطبع($رد);

// الوصول للبيانات
// $رد يحتوي على: {"الحالة": 200, "النص": "...", "البيانات": {...}}
```

### طلب POST (إرسال بيانات)

```havnix
$بيانات = {"الاسم": "عثمان", "العمر": 18};
$رد = ابعت_ل("https://api.example.com/users", $بيانات);
اطبع($رد);
```

### طلب PUT (تحديث بيانات)

```havnix
$بيانات_محدثة = {"الاسم": "عثمان", "العمر": 19};
$رد = حدث_في("https://api.example.com/users/1", $بيانات_محدثة);
```

### طلب DELETE (حذف بيانات)

```havnix
$رد = احذف_من_api("https://api.example.com/users/1");
```

### إرسال Headers مخصصة

```havnix
$هيدرز = {"Authorization": "Bearer TOKEN123", "Content-Type": "application/json"};
$رد = جيب_من("https://api.example.com/protected", فاضي, $هيدرز);
```

### مثال عملي: جلب بيانات الطقس

```havnix
$الطقس = جيب_من("https://api.open-meteo.com/v1/forecast?latitude=15.5&longitude=32.5&current_weather=true");
قول ليهو("حالة الطقس في الخرطوم:");
اطبع($الطقس);
```

## تطبيقات سطح المكتب

هافنيكس تستخدم مكتبة `tkinter` المدمجة في Python لبناء تطبيقات سطح المكتب.

### إنشاء نافذة

```havnix
$نافذتي = نافذة_جديدة("عنوان التطبيق", 500, 400);
```

### إضافة نص ثابت (Label)

```havnix
نص_ثابت($نافذتي, "مرحبا بكم!");
```

### إضافة زر (Button)

```havnix
// زر بسيط
زر($نافذتي, "اضغط هنا");

// زر مع دالة callback
دالة عند_الضغط() {
    قول ليهو("تم الضغط!");
}
زر($نافذتي, "اضغط هنا", "عند_الضغط");
```

### إضافة مدخل نص (Entry)

```havnix
$مدخل = مدخل_نص($نافذتي, "اكتب هنا...");
```

### إضافة مساحة نص (TextArea)

```havnix
$مساحة = مساحة_نص($نافذتي, 10, 40);
// 10 صفوف و 40 عمود
```

### إضافة قائمة منسدلة (Dropdown)

```havnix
$خيارات = ["الخيار 1", "الخيار 2", "الخيار 3"];
$القائمة = قائمة_منسدلة($نافذتي, $خيارات);
```

### إضافة خانة اختيار (Checkbox)

```havnix
$موافق = خانة_اختيار($نافذتي, "أوافق على الشروط");
```

### تشغيل الواجهة

```havnix
شغل_واجهة($نافذتي);
```

### مثال كامل: تطبيق ترحيب

```havnix
// إنشاء النافذة
$نافذتي = نافذة_جديدة("تطبيق الترحيب", 400, 300);

// إضافة العناصر
نص_ثابت($نافذتي, "أدخل اسمك:");
$مدخل_الاسم = مدخل_نص($نافذتي, "الاسم هنا...");

// دالة الترحيب
دالة رحب() {
    قول ليهو("مرحبا!");
}

// زر الترحيب
زر($نافذتي, "رحب بي! 👋", "رحب");

// تشغيل
شغل_واجهة($نافذتي);
```

## الأمثلة الكاملة

### مثال 1: آلة حاسبة

```havnix
دالة حاسبة(عملية, أ, ب) {
    لو (عملية == "+") {
        رجع أ + ب;
    }
    غير كدا لو (عملية == "-") {
        رجع أ - ب;
    }
    غير كدا لو (عملية == "*") {
        رجع أ * ب;
    }
    غير كدا لو (عملية == "/") {
        لو (ب == 0) {
            قول ليهو("خطأ: قسمة على صفر!");
            رجع 0;
        }
        رجع أ / ب;
    }
    غير كدا {
        قول ليهو("عملية غير معروفة");
        رجع 0;
    }
}

$نتيجة = حاسبة("+", 10, 5);
قول ليهو("10 + 5 = $نتيجة");
```

### مثال 2: إدارة مهام

```havnix
$المهام = [];

دالة اضف_مهمة(المهمة) {
    $المهام = اضف($المهام, المهمة);
    قول ليهو("تم إضافة: $المهمة");
}

دالة اعرض_المهام() {
    قول ليهو("=== قائمة المهام ===");
    $عداد = 1;
    لكل $مهمة في $المهام {
        قول ليهو("$عداد. $مهمة");
        $عداد = $عداد + 1;
    }
}

جيب لي اضف_مهمة("تعلم هافنيكس");
جيب لي اضف_مهمة("بناء تطبيق");
جيب لي اضف_مهمة("نشر المشروع");
جيب لي اعرض_المهام();
```

### مثال 3: تطبيق ويب بسيط

```havnix
// جلب بيانات من API وعرضها
$المقالات = جيب_من("https://jsonplaceholder.typicode.com/posts");

قول ليهو("=== آخر المقالات ===");
$عداد = 0;
لكل $مقال في $المقالات {
    لو ($عداد >= 5) {
        وقف;
    }
    اطبع($مقال);
    قول ليهو("---");
    $عداد = $عداد + 1;
}
```

## جدول مرجعي سريع

### الأوامر الأساسية

| الأمر | الوصف | مثال |
|-------|-------|------|
| `قول ليهو()` | طباعة | `قول ليهو("مرحبا")` |
| `اطبع()` | طباعة | `اطبع($متغير)` |
| `اسأل()` | إدخال | `اسأل($الاسم, "اسمك؟ ")` |
| `$اسم = قيمة` | تعريف متغير | `$العمر = 18;` |
| `ثابت $اسم = قيمة` | ثابت | `ثابت $PI = 3.14;` |

### التحكم في التدفق

| الأمر | الوصف |
|-------|-------|
| `لو` | شرط if |
| `غير كدا لو` / `ولو` | شرط elif |
| `غير كدا` | else |
| `طالما` | حلقة while |
| `لكل ... في` | حلقة foreach |
| `تكرار` | حلقة for |
| `وقف` | break |
| `كمل` | continue |

### الدوال

| الأمر | الوصف |
|-------|-------|
| `دالة اسم() { }` | تعريف دالة |
| `جيب لي اسم()` | استدعاء دالة |
| `رجع قيمة` | إرجاع قيمة |

### معالجة الأخطاء

| الأمر | الوصف |
|-------|-------|
| `جرب { }` | try |
| `امسك ($خطأ) { }` | catch |
| `واخيراً { }` / `دايماً { }` | finally |

### الملفات

| الأمر | الوصف |
|-------|-------|
| `اقرأ_ملف("مسار")` | قراءة |
| `اكتب_ملف("مسار", محتوى)` | كتابة |
| `اضف_ملف("مسار", محتوى)` | إضافة |
| `احذف_ملف("مسار")` | حذف |

### قواعد البيانات

| الأمر | الوصف |
|-------|-------|
| `اتصل_قاعدة()` | اتصال |
| `استعلم()` | SELECT |
| `نفذ_استعلام()` | INSERT/UPDATE/DELETE |
| `اقفل_قاعدة()` | إغلاق |

### APIs

| الأمر | الوصف |
|-------|-------|
| `جيب_من(url)` | GET |
| `ابعت_ل(url, data)` | POST |
| `حدث_في(url, data)` | PUT |
| `احذف_من_api(url)` | DELETE |

### الواجهة الرسومية

| الأمر | الوصف |
|-------|-------|
| `نافذة_جديدة()` | إنشاء نافذة |
| `نص_ثابت()` | Label |
| `زر()` | Button |
| `مدخل_نص()` | Entry |
| `مساحة_نص()` | TextArea |
| `قائمة_منسدلة()` | Dropdown |
| `خانة_اختيار()` | Checkbox |
| `شغل_واجهة()` | تشغيل |
